# NexoraDaily

Günlük görev sistemi. Oyuncular her gün 3 rastgele görev alır ve tamamlayarak para kazanır.

## Özellikler
- Her gün 3 rastgele görev
- Görev tipleri: Blok kırma, Mob öldürme
- Ekonomi entegrasyonu
- Tüm görevleri tamamlayınca bonus ödül
- Otomatik günlük sıfırlama (00:00)

## Komutlar
- `/quests` - Günlük görevleri göster

## Görev Ödülleri
- Blok kırma: 200-500₺
- Mob öldürme: 300-600₺
- Değerli maden: 500-1000₺
- Tüm görevler bonus: 500₺
