# NexoraShop

Gelişmiş GUI mağaza sistemi. NexoraEconomy ile entegre çalışır.

## Özellikler
- GUI tabanlı mağaza sistemi
- 6 kategori: Bloklar, Yemek, Aletler, Silahlar, Zırh, Diğer
- Alış/satış fiyatları
- Toplu alım/satım desteği
- Ekonomi entegrasyonu

## Komutlar
- `/shop` - Mağazayı aç
- `/sell` - Elindeki eşyayı sat
- `/sellall` - Envanterindeki tüm satılabilir eşyaları sat

## Kullanım
Mağazada:
- Sol Tık: 1 adet al
- Sağ Tık: 64 adet al
- Shift + Sol Tık: 1 adet sat
- Shift + Sağ Tık: 64 adet sat
