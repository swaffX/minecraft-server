# NexoraAnnouncer

Otomatik duyuru sistemi. Her 5 dakikada bir sohbete bilgilendirici mesajlar gönderir.

## Özellikler
- Otomatik duyurular (5 dakikada bir)
- Komutlar hakkında bilgilendirme
- Sunucu özellikleri tanıtımı
- Sıralı duyuru sistemi

## Duyurular
- /daily komutu
- /shop komutu
- /quests komutu
- /balance komutu
- /baltop komutu
- /pay komutu
- /achievements komutu
- Günlük görevler
- Eşya satışı
- Sunucu IP
- Renk kodları
- Rütbe sistemi

## Ayarlar
- Duyuru aralığı: 5 dakika (6000 tick)
- Oyuncu yoksa duyuru yapılmaz
