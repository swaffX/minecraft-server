# NexoraChat - Gelişmiş Chat Sistemi

## Özellikler
- ✅ Rütbe bazlı chat formatı
- ✅ Anti-spam sistemi (1 saniye cooldown)
- ✅ Renkli yazma desteği (&c, &a vb.)
- ✅ Chat temizleme (/clearchat)
- ✅ Chat susturma (/mutechat)

## Komutlar
- `/clearchat` veya `/cc` - Chat'i temizler
- `/mutechat` veya `/mc` - Chat'i açar/kapatır

## Yetkiler
- `nexorachat.clearchat` - Chat temizleme
- `nexorachat.mutechat` - Chat susturma
- `nexorachat.bypass` - Anti-spam bypass
- `nexorachat.color` - Renkli yazma

## Rütbe Formatı
- **[OWNER]** - Kırmızı, kalın
- **[ADMIN]** - Kırmızı, kalın
- **[MOD]** - Altın, kalın
- **[VIP]** - Aqua, kalın
- **[OYUNCU]** - Gri

## Kurulum
1. JAR dosyasını plugins klasörüne at
2. Sunucuyu yeniden başlat
3. NexoraRanks ile otomatik entegre olur
